# HEI Push-Down Processing Exercises

Collection of PROG3 evaluation exercises focusing on push-down processing with PostgreSQL and Java.

## Exercises

### Exercise 1: American Car Garage Repairs
- Repair statistics for FORD, GMC, DODGE vehicles
- Queries: repair counts, percentages, costs by mechanic

### Exercise 2: Korean Parts Sales
- Parts sales for KIA, HYUNDAI, DAEWOO vehicles
- Queries: parts sold by brand/model, total KIA sales value

### Exercise 3: Best-Selling Parts Tracking
- Sales tracking with dates for KIA and DAEWOO
- Queries: parts sold by brand/model, percentages before cutoff date

## Technologies
- PostgreSQL (ENUM types, aggregations, COALESCE)
- Java with Spring JDBC Template
- Push-down processing pattern
